# Real-Time-Noise-Level-Measurement-System-Hardware-Accelerated-Logarithm-on-Zybo-Z7
Built a real-time noise level measurement system on a Xilinx Zynq-7000 (Zybo Z7-10), sampling analog noise through the onboard XADC and computing sound intensity in dB. Implemented and benchmarked three approaches to the logarithmic dB calculation
